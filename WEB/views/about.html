Preguntas frecuentes

  • ¿Tengo que pagar por utilizar algún servicio?

    • No, puedes utilizar todas las funcionalidades de la aplicación totalmente gratis.

  • ¿Tengo que tener varias cuentas para cada aplicación?

    • No, con la misma cuenta puedes iniciar sesión en la página web y en las aplicaciones de escritorio y móvil.

  • ¿Qué necesito para usar las aplicaciones?

    • Los requisitos para cada aplicación estan indidcados en la página de descargas, pero lo básico para usar la aplicación móvil es necesario tener un dispositivo Android y para la de escritorio un sistema operativo Windows.

  • ¿Cuales son los métodos de pago en la tienda online?

    • En la tienda online el único método de pago disponible es mediante Paypal.
